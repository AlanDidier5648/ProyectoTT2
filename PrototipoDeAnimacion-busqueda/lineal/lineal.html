Explicación General del Algoritmo

El algoritmo de Búsqueda Lineal es uno de los métodos más simples para encontrar un elemento específico en una lista. Este método consiste en recorrer la lista elemento por elemento, comparando cada uno con el valor que se desea encontrar. La búsqueda lineal no requiere que la lista esté ordenada previamente, lo que la hace muy versátil, pero puede ser menos eficiente en listas grandes, ya que su tiempo de ejecución es proporcional al tamaño de la lista.

¿En qué consiste?

El algoritmo de Búsqueda Lineal funciona de la siguiente manera:

  1. Recorrido de la lista: Se empieza desde el primer elemento de la lista y se avanza uno por uno.
  2. Comparación: Cada elemento se compara con el valor buscado:
    • Si el elemento coincide con el valor buscado, la búsqueda termina.
    • Si no coincide, se pasa al siguiente elemento.
  3. Finalización: Este proceso se repite hasta encontrar el elemento o hasta llegar al final de la lista. Si se llega al final sin encontrar el elemento, significa que este no está presente en la lista.

Ejemplo

Lista inicial: [5, 12, 18, 23, 42].
Elemento a buscar: 23.

Primera iteración:
- Se compara el primer elemento 5 con el valor buscado 23.
- Como 5 ≠ 23, se avanza al siguiente elemento.

Segunda iteración:
- Se compara el segundo elemento 12 con el valor buscado 23.
- Como 12 ≠ 23, se avanza al siguiente elemento.

Tercera iteración:
- Se compara el tercer elemento 18 con el valor buscado 23.
- Como 18 ≠ 23, se avanza al siguiente elemento.

Cuarta iteración:
- Se compara el cuarto elemento 23 con el valor buscado 23.
- Como 23 = 23, se ha encontrado el elemento buscado.

El número 23 ha sido encontrado en la posición 4 de la lista.
