ALGORITMO BÚSQUEDA BINARIA


Explicación General del Algoritmo

El algoritmo de Búsqueda Binaria es un método rápido y eficiente para encontrar un elemento específico en una lista ordenada. A diferencia de una búsqueda secuencial, que revisa cada elemento uno por uno, la búsqueda binaria reduce el número de elementos a la mitad en cada paso. Esto se logra mediante una estrategia de divide y vencerás, en la que se compara el elemento objetivo con el elemento central de la lista, descartando la mitad en la que no puede estar el elemento. Debido a esta división constante, la búsqueda binaria es especialmente útil para listas grandes, siempre y cuando estén ordenadas previamente.



¿En qué consiste?

El algoritmo de Búsqueda Binaria funciona de la siguiente manera:

  1. Selección del punto medio: Se elige el elemento que está en el centro de la lista actual. Este valor permite decidir en qué mitad de la lista podría estar el elemento buscado.
  2. Comparación: Se compara el elemento medio con el valor que se está buscando:
    • Si el elemento medio es el valor que se busca, la búsqueda termina.
    • Si el valor buscado es menor, se descarta la mitad derecha de la lista.
    • Si el valor buscado es mayor, se descarta la mitad izquierda de la lista.
  3. Repetición: Este proceso se repite en la mitad restante, eligiendo un nuevo punto medio y descartando la mitad correspondiente, hasta que se encuentra el elemento o se determina que no está en la lista.


Ejemplo

Lista inicial: [3, 8, 12, 20, 27, 33, 42].
Elemento a buscar: 27.

Primera iteración:
- Se selecciona el elemento medio de la lista, que es 20.
- Como 27 > 20, se descarta la mitad izquierda, incluyendo el 20.
- Nota: Es esencial que la lista esté ordenada para asegurar que, al descartar una mitad, el elemento buscado realmente no esté en esa parte.
- Nueva lista: [27, 33, 42].

Segunda iteración:
- En la nueva lista, el elemento medio es 33.
- Como 27 < 33, se descarta la mitad derecha, incluyendo el 33.
- Nueva lista:[27].

Tercera iteración:
- El único elemento restante es 27, que coincide con el valor buscado.

El número 27 ha sido encontrado en la lista.
