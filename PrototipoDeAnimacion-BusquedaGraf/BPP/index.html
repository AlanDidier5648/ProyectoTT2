Explicación General del Algoritmo

El algoritmo de Búsqueda Primero en Profundidad (DFS, por sus siglas en inglés: Depth-First Search) es un método utilizado para recorrer o buscar elementos en estructuras de datos como grafos y árboles. Este algoritmo explora tan lejos como sea posible a lo largo de cada rama antes de retroceder. DFS se basa en una estrategia de "backtracking" (retroceso) y utiliza una pila, ya sea explícita o implícita en el caso de una implementación recursiva. Es útil para descubrir componentes conexos en un grafo, resolver laberintos y generar árboles de decisión.

¿En qué consiste?

El algoritmo de Búsqueda Primero en Profundidad funciona de la siguiente manera:

  1. Inicialización: Se selecciona un nodo inicial para comenzar la búsqueda. Este nodo se marca como visitado.
  2. Exploración: Desde el nodo actual, se selecciona uno de sus nodos adyacentes no visitados y se avanza hacia él.
  3. Retroceso: Si un nodo no tiene nodos adyacentes no visitados, el algoritmo retrocede al nodo anterior para continuar explorando otras ramas.
  4. Finalización: El proceso se repite hasta que todos los nodos alcanzables hayan sido visitados.

Ejemplo

Grafo inicial:
Nodos: {A, B, C, D, E, F, G}.
Arcos: {(A-B), (A-C), (B-D), (B-E), (C-F), (C-G)}.
Nodo de inicio: A.

Paso 1:
- Nodo inicial: A.
- Se visita A y se elige un nodo adyacente no visitado: B.
- Camino actual: [A, B].

Paso 2:
- Nodo actual: B.
- Se visita B y se elige un nodo adyacente no visitado: D.
- Camino actual: [A, B, D].

Paso 3:
- Nodo actual: D.
- D no tiene nodos adyacentes no visitados, se retrocede a B.
- Camino actual: [A, B].

Paso 4:
- Desde B, se elige el siguiente nodo adyacente no visitado: E.
- Camino actual: [A, B, E].

Paso 5:
- Nodo actual: E.
- E no tiene nodos adyacentes no visitados, se retrocede a B, y luego a A.
- Camino actual: [A].

Paso 6:
- Desde A, se elige el siguiente nodo adyacente no visitado: C.
- Camino actual: [A, C].

Paso 7:
- Nodo actual: C.
- Se visita C y se elige un nodo adyacente no visitado: F.
- Camino actual: [A, C, F].

Paso 8:
- Nodo actual: F.
- F no tiene nodos adyacentes no visitados, se retrocede a C.
- Desde C, se elige el siguiente nodo adyacente no visitado: G.
- Camino actual: [A, C, G].

Paso 9:
- Nodo actual: G.
- G no tiene nodos adyacentes no visitados, se retrocede a C, luego a A.
- Todos los nodos han sido visitados.

Orden de visita final: [A, B, D, E, C, F, G].
