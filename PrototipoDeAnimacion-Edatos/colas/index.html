Explicación General del Algoritmo

Una cola es una estructura de datos lineal que organiza elementos en orden secuencial siguiendo el principio FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir. Los elementos se agregan al final de la cola y solo se pueden retirar desde el frente, manteniendo el orden de llegada.

En una cola, las operaciones principales son encolar (añadir un elemento al final) y desencolar (retirar el elemento del frente). Este funcionamiento asegura que los elementos se procesen en el mismo orden en que ingresaron, sin acceso directo a elementos intermedios, lo que hace a la cola simple pero efectiva para gestionar un flujo ordenado de datos.

¿En qué consiste?

Una cola funciona de la siguiente manera:

  1. Inicialización: La cola se crea como una estructura vacía, lista para almacenar elementos en orden secuencial.
  2. Encolar: Cada vez que un elemento nuevo llega a la cola, se coloca al final. Este proceso garantiza que todos los elementos mantengan el orden de llegada.
  3. Desencolar: Para procesar o eliminar un elemento, siempre se toma el que está al frente de la cola. Este paso sigue el principio FIFO (First In, First Out), asegurando que el primer elemento en entrar sea el primero en salir.

Ejemplo

Cola inicial: [].

Primera iteración (Encolar elementos):
- Se añade el elemento A.    Nueva cola: [A].
- Se añade el elemento B.    Nueva cola: [A,B].
- Se añade el elemento C.    Nueva cola: [A,B,C].

Segunda iteración (Desencolar elementos):
- Se retira el  primer  elemento, A.    Nueva cola: [B,C].
- Se retira el  siguiente  elemento, B.    Nueva cola: [C].
- Se retira el  último  elemento, C, dejando la cola vacía [].
