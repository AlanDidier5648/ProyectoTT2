Cuestionario

Responde las siguientes preguntas para desbloquear la animación:

1. ¿Qué principio define el funcionamiento de una cola?




2. ¿Qué sucede durante la operación de encolar un elemento en una cola?




3. En el ejemplo, después de encolar A, B y C, ¿cuál es el orden de los elementos en la cola?




4. ¿Qué ocurre cuando se desencola un elemento en una cola?




5. Si se tienen los elementos [X, Y, Z] en una cola, ¿qué ocurrirá si se encola el elemento W y luego se desencolan dos elementos?
