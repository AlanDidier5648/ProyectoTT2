LISTAS



Explicación General del Algoritmo

Una lista es una estructura de datos lineal que permite almacenar una secuencia de elementos en un orden específico. Cada elemento en una lista tiene una posición o índice que indica su ubicación dentro de la secuencia, comenzando generalmente desde el índice 0. Las listas pueden contener distintos tipos de datos y permiten realizar operaciones como agregar, eliminar o modificar elementos en cualquier posición.


A diferencia de estructuras como las colas o pilas, las listas ofrecen un acceso directo a cualquier elemento mediante su índice, lo que permite una gran flexibilidad para manipular y organizar los datos. Esta estructura es muy versátil y se utiliza en una gran variedad de aplicaciones debido a su capacidad para mantener un orden específico de los elementos y su facilidad de modificación.



¿En qué consiste?

Una lista funciona de la siguiente manera:

  1. Inicialización: Se crea la lista como una estructura vacía o con elementos iniciales en un orden específico.
  2. Acceso a elementos: Cada elemento en la lista tiene una posición (índice), lo que permite acceder a cualquier elemento directamente utilizando su índice. Este acceso directo facilita la manipulación de los datos almacenados.
  3. Agregar elementos: Es posible añadir elementos al final de la lista o en una posición específica dentro de ella. Esto permite mantener o ajustar el orden de los elementos según se necesite.
  4. Eliminar elementos: Los elementos pueden eliminarse especificando su índice o buscando el valor del elemento. Al eliminar un elemento, los índices de los elementos restantes se ajustan para llenar el espacio dejado.
  5. Modificar elementos: Se pueden modificar los elementos de la lista directamente a través de su índice, cambiando su valor sin alterar la posición de los demás elementos.


Ejemplo

Lista inicial: [10,20,30].

  1. Acceso a elementos:
    - Queremos obtener el segundo elemento de la lista, es decir, el que está en la segunda posición.

    En una lista, las posiciones comienzan a contarse desde 0. Esto significa que el elemento en la posición 0 es 10, el elemento en la posición 1 es 20, y el elemento en la posición 2 es 30.

    - Por lo tanto, el segundo elemento está en la posición 1.
    - Resultado: 20.

  2. Agregar elementos:
    - Añadimos el elemento 40 al final de la lista. Esto coloca el 40 después de todos los elementos actuales.
    - Nueva lista: [10, 20, 30, 40].

    - Luego, añadimos el elemento 15 en la posición 1. Esto mueve los elementos siguientes una posición a la derecha para hacer espacio.
    - Nueva lista: [10, 15, 20, 30, 40].

  3. Eliminar elementos:
    - Eliminamos el elemento en la posición 2, que es 20. Al hacerlo, los elementos que estaban después de 20 se mueven una posición hacia la izquierda para llenar el espacio.
    - Nueva lista: [10, 15, 30, 40].

  4. Modificar elementos:
    - Cambiamos el valor del elemento en la posición 1 (que ahora es 15) y lo actualizamos a 25.
    - Nueva lista: [10, 25, 30, 40].
