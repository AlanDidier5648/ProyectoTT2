1. ¿Qué característica principal distingue a las formaciones de otras estructuras de datos?




2. ¿Cómo se puede acceder a los elementos de una formación?




3. Si tienes una formación con los valores [3, 7, 12, 5, 8], ¿qué sucede si intentas acceder al índice 5?




4. ¿Qué operación es necesaria para cambiar el valor en una posición específica de una formación?




5. Supongamos que tienes una formación de tamaño 4: [1, 2, 3, 4]. ¿Qué pasa si intentas añadir un nuevo elemento 5 a esta formación?
