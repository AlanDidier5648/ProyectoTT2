PILAS



Explicación General del Algoritmo

Una pila es una estructura de datos lineal que organiza elementos siguiendo el principio LIFO (Last In, First Out), lo que significa que el último elemento en entrar es el primero en salir. Funciona de manera similar a una pila de platos: el último plato que se coloca en la pila es el primero que se quita.

Las operaciones principales en una pila son apilar (push), que añade un elemento al tope de la pila, y desapilar (pop), que retira el elemento del tope. En una pila, el acceso está restringido solo al elemento en el tope, lo que significa que solo se puede agregar o retirar el elemento en la parte superior, sin acceso directo a los elementos en medio o en el fondo.



¿En qué consiste?

Una pila funciona de la siguiente manera:

  1. Inicialización: Se crea la pila como una estructura vacía, lista para almacenar elementos en orden secuencial.

  2. Apilar (push): Cada vez que un nuevo elemento llega a la pila, se coloca en el tope. Este proceso asegura que el elemento recién añadido esté siempre en la parte superior, accesible para ser retirado primero.

  3. Desapilar (pop): Para retirar un elemento, siempre se toma el que está en el tope de la pila. Este paso sigue el principio LIFO (Last In, First Out), asegurando que el último elemento en entrar sea el primero en salir.



Ejemplo

Pila inicial: []   (vacía).

  1. Apilar elementos (push):

    Se añade el elemento A en el tope de la pila.
    - Pila: [A].

    Se añade el elemento B en el tope de la pila.
    - Pila: [B]
         [A]
    Se añade el elemento C en el tope de la pila.
    - Pila: [C]
         [B]
         [A]

  2. Desapilar elementos (pop):

    Se retira el elemento en el tope, que es C.
    - Pila: [B]
         [A]

    Se retira el siguiente elemento en el tope, que es B.
    - Pila: [A]

    Se retira el último elemento, que es A, dejando la pila vacía.
    - Pila: []   (vacía).
