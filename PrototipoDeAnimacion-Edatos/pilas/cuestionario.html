Cuestionario

Responde las siguientes preguntas para desbloquear la animación:

1. ¿Cuál es la principal característica que define el funcionamiento de una pila?




2. ¿Qué sucede al apilar (push) un nuevo elemento en una pila?




3. Si una pila contiene los elementos [X, Y, Z], ¿qué elemento será retirado primero si se realiza una operación de desapilar (pop)?




4. ¿Qué ventaja tiene el uso de una pila frente a otras estructuras de datos?




5. Si una pila está inicialmente vacía y se apilan los valores 5, 10 y 15, ¿qué pasa si luego se realizan dos operaciones de desapilar?
